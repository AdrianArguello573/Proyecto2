Existen tres tipos de CSS (Cascading Style Sheets) que se pueden vincular a un sitio web HTML, a saber: CSS externo. Se encuentra en un archivo independiente que se vincula a un documento HTML con una etiqueta CSS interno. Es un código CSS que se incrusta en un documento HTML. Para ello, se añade la etiqueta en la sección del documento HTML. CSS en línea. Permite aplicar reglas CSS a elementos específicos. Para ello, se aplican estilos directamente en la etiqueta de apertura de cualquier elemento HTML a través del atributo “style”. El CSS es un lenguaje basado en reglas que permite definir grupos de estilos que se aplicarán a elementos o grupos de elementos de una página web.
