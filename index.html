CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para describir la presentación de un documento escrito en un lenguaje de marcado, como HTML. Introducido por primera vez en 1996, CSS permite a los desarrolladores web separar el contenido de un sitio web de su diseño visual, facilitando la creación de páginas web atractivas y funcionales.
